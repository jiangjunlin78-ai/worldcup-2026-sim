World Cup 2026 Analyzer
A pure client-side data simulation tool for exploring probability analysis and scoring computation across 48 teams in the 2026 FIFA World Cup.

Features
Interactive data table with sort, search, and batch random seed
Real-time computation: Score = Points × 100 ÷ Probability(%)
All data persisted in browser localStorage — zero server dependencies
Single static HTML file, no build step required
Tech Stack
Vanilla HTML/CSS/JS. No framework, no backend, no analytics.

Disclaimer
This is a mathematical simulation tool. All data is user-provided or auto-generated for demonstration. No real currency, betting, or financial transactions are involved. For entertainment and technical reference only.

Deploy
